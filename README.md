# 📱 Enhancing Smishing Detection Using Deep Learning

A Deep Learning based Smishing Detection System that identifies malicious SMS messages using CNN, LSTM, and Bidirectional LSTM models.

## 🎯 Project Objective

To improve smishing detection accuracy while reducing false positives using advanced deep learning techniques and natural language processing.

## 🚀 Key Features

- SMS Text Preprocessing using NLTK
- TF-IDF Feature Extraction
- LSTM Model
- BiLSTM Model
- CNN + LSTM Hybrid Model
- CNN + LSTM + BiLSTM Extended Model
- Flask Based Web Application
- High Accuracy (98.96%)

## 🛠 Technologies Used

- Python

- Keras
- Scikit-Learn
